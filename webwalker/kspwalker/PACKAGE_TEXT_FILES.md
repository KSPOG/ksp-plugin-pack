# KSP Walker package text files

These text files are intentionally kept inside:

```text
net.runelite.client.plugins.microbot.kspwalker
```

Files:

```text
kspwalker_transports.txt
kspwalker_teleports.txt
```

Runtime copies are created/loaded from:

```text
%USERPROFILE%\.kspwalker\kspwalker_transports.txt
%USERPROFILE%\.kspwalker\kspwalker_teleports.txt
```

The package-local files are the editable/default examples that should travel with the plugin source package.
